post /generate_billing_statement
Request to generate account billing statement. (generateBillingStatementPost)
The request to generate billing statment for an account. Generated billing statement will be in format of pdf. Client needs to fetch the result file using returned url. Generating of billing statement is supported only on account level.

This API call consumes the following media types via the Content-Type request header:
