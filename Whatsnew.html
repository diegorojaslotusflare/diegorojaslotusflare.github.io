post /add_batch
Batch of msisdns are added to number management inventory (addBatchPost)
Adds a batch of msisdns. Msisdns which already exist in the number management inventory will not be added again.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body add_batch_body (required)
Body Parameter —
example: { "summary" : "Add 3 numbers in a batch", "value" : { "batch" : [ "123456789012345", "234567890123456", "34567890123456" ] } }

Return type

batch_result

Example data

Content-Type: application/json
{
  "server_time" : 1545667505,
  "invalid_numbers" : [ "123456789012345", "234567890123456" ],
  "added_count" : 0.8008281904610115
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Request successful batch_result

400

Invalid request. This may be due to invalid JSON, or invalid/missing fields in the request body. failed_request_body

post /download
Downloads various numbers with given status from number management inventory. (downloadPost)
Downloads various numbers with given status in number management inventory in a streaming fashion. Currently inception, reserved and available status are supported. The response might contain duplicates and the caller is responsible for removing them. This api is only available as an operator facing call.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body download_body (required)
Body Parameter —
example: { "summary" : "Get all numbers with inception status with each page containing 2 numbers", "value" : { "status" : 1, "page_size" : 50 } }

Return type

array[array[Integer]]

Example data

Content-Type: application/json
[ [ 0, 0 ], [ 0, 0 ] ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

post /request
Request a set of msisdns from number management inventory (requestPost)
Retrieves a set of available msisdns from number management inventory. If specific requirements have been specified in the request, the msisdns returned would be filtered to match the requested criteria.

download_body Up

bucket (optional)
Integer The numbers with given status from number management inventory INCEPTION = 1, All numbers that were ever added to the inventory AVAILABLE = 2, All numbers that are available to be reserved RESERVED = 3 , All numbers that are currently reserved but not assigned COOLDOWN = 4, All numbers that are current in cooldown status i.e numbers that got freed up after termination
paze_size (optional)
Integer The number of results to return per page. If not specified default page size(1000) is used.
max_results (optional)
Integer The maximum results to be returned per status. If not specified all results are returned. The actual number of entities returned would be greater than max results at the most by a page_size in case max_size % page_size ~= 0
