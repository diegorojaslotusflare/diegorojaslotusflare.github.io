post /get_conversations
Gets a list of conversations given id or a date range (getConversationsPost)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body get_conversations_body (required)
Body Parameter —
example: { "chat_id" : "2294216f-9f98-46e9-8e77-9cdd57c814d0", "date_from" : "2021-09-10T00:00:00.000Z", "date_to" : "2021-09-10T23:00:00.000Z" }
