post /add_asset
Store asset and get signed s3 PUT url (addAssetPost)
The asset we want to add must match the asset type we support on the backend. The API must also provide the identifier we want to add the asset for and can be either user id, account id or subscriber id. The output of the API will be signed s3 PUT url, that the caller can then use to upload the image to.
