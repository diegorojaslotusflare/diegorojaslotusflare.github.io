page_size Up

This is the requested maximum number of results to be returned when requesting paged results.

paging_state Up

This is a state value returned when paging of results is requested. Each subsequent request for a page of results should use the previous response's paging_state. If no paging_state is returned in a response, that means there is no more data to retrieve.

unit_numbers Up

Comma-separated list of valid unit number strings (e.g. 1,2,3,4,5A,5B,PH). Represents unit numbers that are valid for a given address. This will allow the website to avoid proceeding with the buyflow for an invalid unit number. Null or empty field value will mean all unit numbers are valid.
