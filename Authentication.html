API Authentication

Token 

Token used by External Partners to Authenticate with the LotusFlare Server.

HTTP Header

We will use the Authorization HTTP header for authentication. The value of the Authorization header will contain the string “Bearer”, followed by a JWT token, as shown here:

Authorization: Bearer <JWT token>


The JWT token can be constructed as follows:


Base64UrlEncoded(JWT Headers).Base64UrlEncoded(JWT Payload).Base64UrlEncode(HS256)


The JWT token is constructed by concatenating Base64UrlEncoded JWT Headers, Payload and HMAC Computed (See https://jwt.io/) with . as a separation character.


The full JWT specification is documented in RFC7519: https://tools.ietf.org/html/rfc7519
