address_map Up

We couldn't predict exactly what fields address_parts object will contain so it's left to be dynamic. In general, it should at least contain the field <strong>address</strong>. For some operators, we might have special requirements when adding an address. Only tenants and sub tenants will be able to see tax_area_id.
