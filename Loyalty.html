post /get_registrations
Gets a list of all Reward Programs, with subscriber registration and eligibility statuses. (getRegistrationsPost)

Request headers

Return type

  "reward_programs" : {
    "key" : {
      "reward_program_id" : "b1c9f50e4d32459da4c3563ededf6d90",
      "is_registered" : true,
      "eligibility" : {
        "eligible" : true,
        "eligibility_reason" : 0
      }
    }
  },

post /set_registration
Sets the user's registration state for the given reward program. If no reward program is specified and there is only one active reward program, this method will apply to that program. (setRegistrationPost)
